# FE-HandTris

- [Next.js](https://nextjs.org/)
- [TensorFlow.js](https://www.tensorflow.org/js)

## 커밋 메시지 규칙

- **[🛠️ CREATE]**: 새로운 파일이나 프로젝트 생성
- **[🪽 FEAT]**: 새로운 기능 추가
- **[🎨 UI/UX]**: 사용자 인터페이스 및 사용자 경험 개선
- **[🐛 FIX]**: 버그 수정
- **[🧹 REFACTOR]**: 코드 리팩토링
- **[📝 DOCS]**: 문서 작성 및 수정
- **[🔧 CONFIG]**: 설정 파일 수정
- **[♻️ CHORE]**: 기타 자잘한 작업
